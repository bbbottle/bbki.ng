# 与或非禁区


**0. 河流紀事 · 鄧獻誌**

河流用一生梳理 \
地球的表情 \
沿岸收集人類的喜怒哀樂 \
檔案堆滿了海底的家

**1. 古老的小区**

我搬进 \
一个古老的小区的胃里

堆积的法国梧桐树叶 \
和我的过往 \
加重了它消化不良的症状

北方的冷空气穿过冲积平原 \
匍匐在小区旁的江面休息 \
玩闹

手拉着手的山脉远在南边等候 \
胃壁从此有水珠渗出

我的浴巾不再干燥

**2. 信**

被子折叠成信封 \
身体干瘪成信纸

睡梦邮递心事。

**3. 梦里的事**

梦里一群恒星散落在 \
外婆家屋后的草丛

梦里在摩天大楼上空蛙泳

梦里变成凶杀犯被通缉

梦里牙齿全部脱落 \
一点也不痛
