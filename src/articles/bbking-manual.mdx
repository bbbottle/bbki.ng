## 快捷键

在浏览本站的过程中，需要频繁在路由间跳转，除了点击超链接的方式，也可以直接按下对应的快捷键进行跳转。下表是快捷键说明。

| 快捷键 | 含义       | 描述                           |
| :----- | :--------- | :----------------------------- |
| `h`    | help       | 跳转到本文                     |
| `i`    | index      | 跳转到首页                     |
| `c`    | content | 跳转到目录页                     |
| `shift + t`    | tags        | 跳转到标签列表                 |
| `f`    | forward    | 前进一页                       |
| `b`    | back       | 后退一页                       |
| `s`    | source     | 查看本站源代码                 |

跳转到目标页面后，按下 tab 可方便地进一步选择目标链接。
