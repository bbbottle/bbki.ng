# 导航快捷键

在浏览本站的过程中，需要频繁在路由间跳转，除了用鼠标点击超链接的方式，也可以直接按下对应的快捷键进行跳转。下表是快捷键说明。

| 快捷键 | 含义       | 描述                           |
| :----- | :--------- | :----------------------------- |
| `h`    | help       | 跳转到本文                     |
| `i`    | index      | 跳转到首页                     |
| `e`    | extensions | 跳转到扩展名（媒体类型）选择页 |
| `t`    | txt        | 跳转到文本列表                 |
| `p`    | png        | 跳转到图片列表                 |
| `f`    | forward    | 前进一页                       |
| `b`    | back       | 后退一页                       |

跳转到目标页面后，按下 tab 可方便地进一步选择目标链接。
